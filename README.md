# Cosmos - Guerra de Espirógrafos

Este projeto é uma simulação gráfica massiva e um jogo multiplayer competitivo baseado em curvas matemáticas complexas (Espirógrafos).

## Desenvolvido por
**Miller** (Millerium)

## Recursos
- **Motor Shader Avançado:** Renderização de 256 espécimes matemáticos em tempo real usando WebGL.
- **IA Experimental:** Processamento de linguagem natural simples para configurar parâmetros de curvas.
- **Multiplayer em Tempo Real:** Sincronização via Supabase com mecânicas de combate (Dash, roubo de massa e dreno de fótons).
- **Laboratório Criativo:** Edição granular de parâmetros geométricos e físicos.

## Tecnologias
- HTML5 / Vanilla JavaScript
- GLSL (WebGL Shaders)
- Supabase (Backend Real-time)
- Google Fonts (Outfit)
